# Ruta de estudio

## Debes poder explicar

- Objetivo del juego.
- Prototipo, llamada y definición.
- Diferencia entre `=`, `==`, `&&` y `||`.
- Uso de `do while`, `while` y `for`.
- Uso de `% 10` y `/ 10`.
- Validación de cinco dígitos diferentes.
- Menú de turno: intentar, revelar, pedir pista o rendirse.
- Cálculo del puntaje.
- Desafío final.

## Método

1. Leer una sección.
2. Cerrar el documento.
3. Explicarla con tus palabras.
4. Señalar el código correspondiente.
5. Ejecutar un ejemplo.
