# Simulacros

Cada simulacro vale `100` puntos:

| Parte | Puntaje |
| :--- | ---: |
| Pregunta 1 | 33 |
| Pregunta 2 | 33 |
| Modificación | 34 |

## Simulacro 1

1. Explica `% 10` y `/ 10` con `58274`.
2. ¿Por qué la ayuda inicial mejora la experiencia?
3. Permite hasta `25` intentos.

## Simulacro 2

1. Explica la diferencia entre `while` y `for`.
2. Explica cómo detectas repeticiones sin arrays.
3. Cambia el primer secreto a `68139`.

## Simulacro 3

1. Explica cómo se calcula el puntaje.
2. Con secreto `58274` e intento `12345`, calcula la respuesta.
3. Haz que cada pista reste `10` puntos.

## Simulacro 4

1. Explica prototipo, llamada y definición.
2. Explica por qué `87452` es la clave final de `58274`.
3. Cambia la ayuda inicial a la posición `2`.
